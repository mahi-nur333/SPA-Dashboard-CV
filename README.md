## ✨ Features

- 🗂️ **Single Page Application** — no page reloads, instant navigation
- 📱 **Responsive Design** — works on desktop and mobile
- 🔵 **Collapsible Sidebar** — expand/collapse with toggle button, tooltip on collapse
- 🎨 **Consistent Theme** — MUI primary blue throughout all pages
- ⚡ **React useState** — page routing and sidebar state management
- 🔗 **Clickable Links** — DOI, GitHub, LinkedIn, ORCID, Portfolio
